CLAUDE MEMORY SYSTEM - PAQUETE UNIVERSAL
=========================================

Sistema de memoria para Claude Code, listo para usar en cualquier proyecto.

CONTENIDO:
- Sistema load/save/check/commit + auditorias (security, operational)
- Agente doc-changelog (Haiku)
- Templates para PROJECT, STATE, CHANGELOG, GUARDRAILS, CODEBASE-MAP, FEATURE-CHANGELOG
- Scripts de enforcement (findMany+take, auth guards, uploads, metadata, scaffold)
- Documentacion: README, INFO
- Mapa de proyectos (PROJECT-MAP.md) e infraestructura (INFRA.md)

ARCHIVOS INCLUIDOS:
- 2 docs principales (README, INFO)
- 8 comandos (load, save, check, commit, kickstart-nextjs, cleanup-boilerplate, security-audit, operational-audit)
- 1 agente (doc-changelog)
- 6 scripts de enforcement (check-all, check-findmany-take, check-auth-guards, check-upload-validation, check-page-metadata, check-scaffold)
- 7 templates .planning/ (PROJECT, STATE, CHANGELOG, GUARDRAILS, CODEBASE-MAP, FEATURE-CHANGELOG)
- 2 tiers CLAUDE.md (Full SaaS, Simple)

CARACTERISTICAS PRINCIPALES:
- STATE.md: Estado conciso, se sobrescribe cada /save (max 120 lineas)
- CHANGELOG.md: Historial completo, se acumula cada /save
- GUARDRAILS.md: Errores conocidos (primer filtro)
- CLAUDE.md: Reglas permanentes (escaladas de GUARDRAILS cuando se repiten)
- CODEBASE-MAP.md: Indice del sistema (donde esta cada pieza)
- /check: Auditoria de calidad adaptativa (solo checks relevantes)

FLUJO:
/load -> Lee STATE -> Resume -> Trabaja
/check -> Detecta que cambio -> Checks relevantes -> Reporte
/save -> /check -> Errores -> GUARDRAILS/CLAUDE.md -> STATE -> CHANGELOG -> Commit
/commit -> Commit intermedio durante la sesion

ESCALACION DE ERRORES:
Error nuevo -> GUARDRAILS.md
Error repetido -> CLAUDE.md (regla permanente)

VALIDADO: Multiples auditorias de calidad, produccion, deploy en Coolify

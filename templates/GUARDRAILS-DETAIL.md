---

## Cómo usar este archivo

1. **Cuando se comete un error**: Buscar la sección correspondiente por anchor
2. **Para entender el "por qué"**: Leer contexto completo y ejemplos
3. **Para mejorar el índice**: Usar el contexto para hacer el check más específico
